공통 패턴
- Rendered preview (htmlpreview): https://htmlpreview.github.io/?https://raw.githubusercontent.com/edupsymj-spec/homekma/main/<파일명>
- GitHub Pages (if enabled): https://edupsymj-spec.github.io/homekma/<파일명>
- Raw source: https://raw.githubusercontent.com/edupsymj-spec/homekma/main/<파일명>
